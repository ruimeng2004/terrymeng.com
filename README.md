# Terry Meng Personal Website

Static academic homepage for `terrymeng.com`.

## Local preview

Open `index.html` directly in a browser, or run a simple local server:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.

## Deployment target

Recommended next step: deploy this repository to Cloudflare Pages, then connect
the custom domain `terrymeng.com`.
